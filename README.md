# Subscription-Business-Health-Dashboard
## Project Title / Headline
Subscription Business Health Dashboard: Customer Churn & Retention Analysis
## Short Description / Purpose
This project analyzes a subscription-based customer dataset to evaluate churn behavior, customer retention patterns, and revenue performance. Using data analysis and interactive dashboards, the project identifies factors influencing customer churn and highlights segments that present potential revenue risk. The dashboard enables stakeholders to monitor subscription health and make data-driven retention decisions.
## Tech Stack
SQL – Data exploration and business queries
Power BI – Dashboard creation and data visualization
DAX – KPI calculations including churn rate, retention rate, MRR, and CLV
Power Query – Data cleaning and transformation
Excel / CSV dataset – Data storage and preprocessing
## Data Source
Dataset: Subscription Customer Dataset from Kaggle.
The dataset contains subscription information for a telecom/service-based business.
Dataset size: 7,000+ customer records
Key columns include:
Customer ID
Contract Type
Tenure (months)
Monthly Charges
Total Charges
Churn Status
Customer Demographics
This dataset is commonly used for customer churn analysis and retention modeling.

## Features / Highlights
## 1. Business Problem
Subscription-based businesses rely heavily on customer retention for stable revenue growth. However, identifying customers likely to churn is difficult without structured analysis.
High churn rates can lead to:
Revenue loss
Increased customer acquisition costs
Reduced customer lifetime value
Understanding which customers are at risk of leaving and why is essential for designing effective retention strategies.
## 2. Goal of the Dashboard
The dashboard aims to provide a comprehensive view of subscription business health, enabling stakeholders to:
Monitor overall churn rate
Identify high-risk customer segments
Analyze churn patterns by contract type and tenure
Track revenue metrics such as Monthly Recurring Revenue (MRR)
Understand customer lifecycle behavior

This visual analyzes churn behavior across different contract types such as:
Month-to-Month

It helps identify which contract types are more prone to churn.
## 4️. Customer Tenure vs Churn
This visual explores how churn varies based on customer tenure.

## 5️.Cohort Retention Analysis
A cohort heatmap visual tracks customer retention across time periods, helping identify patterns in long-term customer retention.
This is useful for understanding customer lifecycle performance.

## Business Insights
Key insights derived from the analysis include:
Customers with month-to-month contracts exhibit significantly higher churn rates compared to long-term contracts.
Customers with short tenure periods show higher churn probability, indicating early lifecycle churn risk.
Long-term contract customers demonstrate higher retention stability.
Monitoring churn trends can help businesses identify revenue-at-risk segments.

The insights generated from this analysis can help businesses:
